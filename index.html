📊 Conectar ao Google Sheets

Configure uma vez — os dados de todos os celulares vão para a mesma planilha.

Passo a passo (5 minutos)

1

Acesse sheets.google.com e crie uma planilha nova. Copie o ID da planilha da URL:
docs.google.com/spreadsheets/d/ID_AQUI/edit

2

Acesse script.google.com → Novo projeto. Cole o código do arquivo codigo_apps_script.gs (incluído no ZIP). Coloque o ID da planilha na variável SHEET_ID.

3

No Apps Script: Implantar → Novo deployment → Tipo: App da Web → Executar como: Eu → Quem tem acesso: Qualquer pessoa → Implantar

4

Copie a URL do deployment (começa com https://script.google.com/macros/s/...) e cole abaixo

Cole a URL completa gerada no passo 4
